## Описание

Веб-приложение является курсовой работой по предмету "Технология разработки программного обеспечения". Основной целью создания была практика разработки ПО с использованием возможностей Spring и ЯП Java.

## Технологии

- Java
- PostgreSQL
- Spring Security
- HTML, CSS, JavaScript
